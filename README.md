# Handyman

### Tech
* GoLang
* GraphQL
* Postgresql
* Docker
* pgadmin

### Installation

Handyman requires `Docker`,`Docker-compose` to run.

Install the dependencies and devDependencies and start the server.

```sh
$ docker-compose up
```

Open [this address](http://localhost:8080/) in your preferred browser.
```sh
http://localhost:8080/
```
